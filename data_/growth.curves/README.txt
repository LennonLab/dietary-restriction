COntents
1> In the folders are raw OD data; growth curve figures; and summarized growth parameter data. Growth parameters were calculated used an R script from https://github.com/LennonLab/Growth_Curves
2> Day 150 was preliminary data. Day 275 phenotyped the final evolved lines (as well as ancestors).
3> the SPSS files were used to generate preliminary figures.
4> The .jpg files are preliminary figures
5> The python scripts convert data formatted by the plate reader into a simpler spreadsheet. The n.no script converted the raw values into values proportional to the intitial OD value, while the no.transform script does not transform the raw OD data.
